# Orient-PadTemplate
`Orient-PadTemplate`项目旨在为`Android Pad`项目搭建提供参考方案。 

## 一、技术选型

- 语言：`Java`或者`Kotlin`

- 数据库框架：`GreenDao`
- 数据观察库：`RxJava 2`
- 生命周期处理库：`Lifecycle`
- 网络请求库：`Retrofit 2`
- 控件绑定库：`Butterknife`
- 图片处理：`Glide`
- 设计模式：`MVP`
